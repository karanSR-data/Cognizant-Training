# Microservices with Spring Boot 3 and Spring Cloud

## Introduction

Microservices architecture divides a large application into small, independent services. Each service performs one specific task and communicates with other services using APIs.

Spring Boot 3 and Spring Cloud provide tools to build, manage, and deploy microservices efficiently.

---

## Topics Covered

- Introduction to Microservices
- Monolithic vs Microservices
- Spring Boot 3
- Spring Cloud
- REST API Communication
- Service Discovery
- API Gateway
- Configuration Management

---

## Advantages

- Easy to scale
- Independent deployment
- Better fault isolation
- Faster development
- Easier maintenance

---

## Learning Outcomes

- Understand microservices architecture.
- Build services using Spring Boot.
- Connect multiple services.
- Learn Spring Cloud components.
