# NUnit and Moq

## Overview

NUnit is a unit testing framework for .NET applications, while Moq is a popular mocking framework used to simulate dependencies during testing. Together, they help developers create reliable, maintainable, and testable applications.

## Topics Covered

* Introduction to Unit Testing
* NUnit Framework
* Test Fixtures
* Assertions
* Test Cases
* Mocking with Moq
* Verifying Method Calls
* Dependency Isolation

## Concepts Learned

* Writing automated unit tests
* Testing application components independently
* Creating mock objects for dependencies
* Improving software quality through testing

## Repository Contents

This folder will include:

* NUnit test examples
* Moq examples
* Sample test cases
* Practice exercises
* Notes

## Learning Outcome

Learned the fundamentals of unit testing using NUnit and dependency mocking using Moq to build reliable and maintainable software applications.
