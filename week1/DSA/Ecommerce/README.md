# Exercise 2 - E-commerce Search

## Big O Notation

Big O notation measures the efficiency of an algorithm as input size grows.

## Search Complexity

| Algorithm | Best | Average | Worst |
|-----------|------|----------|--------|
| Linear Search | O(1) | O(n) | O(n) |
| Binary Search | O(1) | O(log n) | O(log n) |

## Conclusion

Binary Search is faster for large sorted datasets.

Linear Search is suitable for small or unsorted datasets.
