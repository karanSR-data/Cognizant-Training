# Week 2

## Overview

The second week of the Cognizant Training Program focused on the Spring Framework ecosystem and enterprise Java application development. The sessions introduced the fundamentals of Spring Core, dependency management using Maven, Object-Relational Mapping (ORM) with Hibernate, and database interaction using Spring Data JPA with Spring Boot.

---

## Topics Covered

### 1. Spring Core

* Introduction to Spring Framework
* Inversion of Control (IoC)
* Dependency Injection (DI)
* Spring Beans
* Bean Lifecycle
* Bean Scopes

---

### 2. Maven

* Introduction to Maven
* Project Structure
* POM (Project Object Model)
* Dependency Management
* Maven Build Lifecycle
* Plugins

---

### 3. Hibernate

* Introduction to ORM
* Hibernate Architecture
* Entity Mapping
* Configuration
* CRUD Operations
* Session and SessionFactory

---

### 4. Spring Data JPA with Spring Boot

* Spring Boot Integration
* Repository Interfaces
* CRUD Repository
* JPA Repository
* Entity Relationships
* Database Operations

---

## Repository Contents

This week's folder will include:

* Learning notes
* Spring Core examples
* Maven project configurations
* Hibernate implementations
* Spring Data JPA CRUD applications

---

## Learning Outcome

By the end of Week 2, I developed a strong understanding of the Spring Framework, dependency management using Maven, object-relational mapping with Hibernate, and database access using Spring Data JPA within Spring Boot applications.
