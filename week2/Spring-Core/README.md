# Spring Core

## Overview

Spring Core is the foundation of the Spring Framework. It provides essential features such as Inversion of Control (IoC) and Dependency Injection (DI), enabling developers to build modular, loosely coupled, and maintainable Java applications.

## Topics Covered

* Introduction to Spring Framework
* Inversion of Control (IoC)
* Dependency Injection (DI)
* Spring Beans
* Bean Scopes
* Bean Lifecycle
* Configuration using Annotations

## Concepts Learned

* Creating and managing Spring Beans
* Understanding the IoC Container
* Implementing Dependency Injection
* Managing Bean Lifecycle
* Building loosely coupled applications

## Repository Contents

This folder will contain:

* Spring Core examples
* Dependency Injection implementations
* Bean configuration examples
* Practice exercises
* Learning notes

## Learning Outcome

Gained a solid understanding of the core principles of the Spring Framework and how dependency injection simplifies enterprise application development.
