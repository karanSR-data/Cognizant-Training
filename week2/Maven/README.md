# Maven

## Overview

Maven is a build automation and dependency management tool used for Java projects. It simplifies project configuration, manages external libraries, and standardizes the build process.

## Topics Covered

* Introduction to Maven
* Maven Project Structure
* POM (Project Object Model)
* Dependency Management
* Build Lifecycle
* Maven Plugins

## Concepts Learned

* Creating Maven projects
* Managing project dependencies
* Understanding the build lifecycle
* Configuring plugins
* Packaging Java applications

## Repository Contents

This folder will contain:

* Maven project examples
* Sample `pom.xml` files
* Dependency management examples
* Notes and practice exercises

## Learning Outcome

Learned how Maven streamlines Java project development through standardized project structures, automated builds, and efficient dependency management.
