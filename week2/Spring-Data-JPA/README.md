# Spring Data JPA

## Overview

Spring Data JPA simplifies database access by providing repository interfaces that reduce boilerplate code and integrate seamlessly with Spring Boot and Hibernate.

## Topics Covered

* Introduction to Spring Data JPA
* Repository Interfaces
* CRUD Repository
* JPA Repository
* Custom Queries
* Entity Relationships
* Integration with Spring Boot

## Concepts Learned

* Creating repository interfaces
* Performing CRUD operations
* Working with entities
* Writing custom query methods
* Connecting Spring Boot applications to relational databases

## Repository Contents

This folder will contain:

* Repository examples
* Entity classes
* CRUD applications
* Custom query examples
* Learning notes

## Learning Outcome

Learned how Spring Data JPA simplifies data persistence by providing a clean repository-based approach for interacting with relational databases in Spring Boot applications.
