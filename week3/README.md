# Week 3

## Overview

The third week of the Cognizant Training Program focused on developing RESTful web services using Spring Boot 3. The sessions introduced the fundamentals of REST architecture, HTTP methods, request handling, and building scalable APIs using the Spring Framework.

---

## Topics Covered

### 1. Spring REST using Spring Boot 3

* Introduction to REST Architecture
* Spring Boot REST Application Setup
* REST Controllers
* HTTP Methods (GET, POST, PUT, DELETE)
* Request Mapping
* Path Variables
* Request Parameters
* Request Body and Response Body
* ResponseEntity
* CRUD Operations
* JSON Request and Response Handling

## Learning Outcome

By the end of Week 3, I gained practical knowledge of building RESTful APIs using Spring Boot 3, handling HTTP requests and responses, implementing CRUD operations, and structuring enterprise-grade backend applications following REST principles.
