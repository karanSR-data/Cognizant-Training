## Objective

This exercise demonstrates how to connect a Spring Boot application with a MySQL database using Spring Data JPA and Hibernate.

- Spring Data JPA
- Hibernate
- MySQL Integration

- Repository Pattern
- Service Layer
- CRUD Operations

- CountryService.java

- country.sql
